# Raw Sources

This directory is the immutable source layer for the TurboQuant-eval wiki.

Rules:

- Put newly captured source material here.
- Do not edit an existing source file after it is added.
- If a source needs correction, add a new file with a new date or suffix.
- Large binaries or images should go under `raw/assets/`.

Suggested naming:

- `YYYY-MM-DD-short-title.md`
- `YYYY-MM-DD-short-title.txt`
- `YYYY-MM-DD-short-title.json`

Current status:

- `raw/legacy-memory/` now contains frozen canonical snapshots of the two key historical notes that were originally written under `memory/`.
- `raw/benchmarks/` now contains frozen benchmark result snapshots used by the current ANN comparison pages.
- `memory/` remains a legacy working-note area, not the canonical source layer.
- New canonical source captures should be added here.
